# Exploratory-Data-Analysis-Business-Intelligence
Exploratory Data Analysis and Business Intelligence project using Python, SQLite, and dashboard visualizations

## Objective
This project aims to uncover patterns, trends, and relationships in retail sales data and answer business questions using SQL and Python visualizations.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- SQLite
- GitHub

## Project Steps
1. Descriptive Statistics & Univariate Analysis
2. SQL for Business Questions
3. Multivariate Analysis & Correlation
4. Static Dashboard Mock-up

## Files in This Repository
- `eda_bi_project.py` - Main Python project file
- `data/retail_sales.csv` - Dataset used for analysis
- `sql/business_queries.sql` - SQL queries for business questions
- `outputs/` - Query results, charts, and dashboard image

## Business Questions
1. What are the top 5 products by revenue?
2. What is the monthly sales trend?
3. Which product category generates the highest revenue?
4. What is the average customer age by category?
5. Which payment method is used most?
6. Which city gives the highest sales?

## Output
The project generates:
- Descriptive statistics
- Univariate charts
- SQL query outputs
- Correlation heatmap
- Scatter plot
- Dashboard mock-up

## How to Run
```bash
pip install pandas matplotlib seaborn
python eda_bi_project.py
```
